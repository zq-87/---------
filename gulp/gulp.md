#### gulp的安装：

> 如果你先前将 gulp 安装到全局环境中了，请执行 `npm rm --global gulp` 将 gulp 删除再继续以下操作。

```js
// 全局安装
npm install gulp -g
// 局部安装
npm install --save-dev gulp
npm install --save-dev gulp@3.9.1
```
